# Client 

Download Expo Go on your phone
Ensure that your devices are connected to the same network

To set up this project:
```bash
npm install --legacy-peer-deps
npx expo start
```

To update the SDK:
```bash
npm install expo@{version number}
npx expo install --check
npx expo start
```
